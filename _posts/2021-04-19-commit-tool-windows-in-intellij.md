## 목적
- 간혹 실수로 Commit 창을 닫거나 비활성화 시켰을 때 잊어버리곤 해서 정리

## Commit Tool Windows?
- 깃으로 관리되는 작업한(추가, 수정, 삭제) 파일을 확인 할 수 있다.
- 하단에 커밋메세지를 작성하여 바로 커밋할 수 있는 메뉴(창) 중 하나이다.
- 나중에 설명할 Shelf, 파일 충돌 해결 등 정말 자주 사용하는 메뉴(창)이다.

![commit](/images/commit.png)



## 활성화 방법
- intellij 설정에서 Version Control > Commit > Use non-modal commit interface 체크

![commit active](/images/commit_tool_window.png)

## 미세팁
- keymap 중 **command + 0 을 commit** 으로 추가하여 사용 중 이다.
